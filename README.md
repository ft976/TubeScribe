# Notestube: Professional Academic Lecture Analyst

Notestube is a powerful academic tool designed to transform long, complex YouTube lectures into structured, mastered study materials. By integrating multi-engine AI synthesis, Notestube helps students, researchers, and lifelong learners maximize knowledge retention and minimize study time.

## 🎓 Core Features

### 1. Multi-Engine Lecture Analysis
Notestube leverages multiple AI engines to analyze video content, ensuring depth and accuracy in the extracted information. It automatically summarizes content, generates key takeaways, and identifies core academic concepts.

### 2. Interactive Academic Workspace
Notestube includes an advanced AI Chat assistant specifically tuned for academic inquiry.
*   **Clarification & Context:** Ask the AI questions about topics discussed in the lecture.
*   **Logical Reasoning:** Check your understanding of complex arguments or theories.
*   **Study Assistance:** Engage with the chatbot to create custom quizzes or summaries on specific segments of the material.

### 3. Integrated Mastery Library
Never lose track of your hard work. Notestube features a dedicated persistent library where you can:
*   **Save Summaries:** Store your processed lecture notes securely.
*   **Review History:** Easily revisit past lectures and notes.
*   **Manage Content:** Delete or wipe your academic history whenever necessary.

### 4. Structured PDF Generation
Convert your learning materials into professional-grade study guides and lecture manuals (PDFs) with a single click, perfectly structured for offline reading and review.

---

## 🚀 How it Works

1.  **Input:** Paste the URL of the YouTube lecture you wish to analyze.
2.  **Analysis:** The system processes the audio/transcript and synthesizes findings through its academic engine.
3.  **Review:** Examine the generated high-level notes, summaries, and key takeaways.
4.  **Engage:** Utilize the AI Workspace to dive deeper into specific topics.
5.  **Master:** Retain your findings by saving them to your Mastery Library.
6.  **Export:** Compile your study materials into a professional PDF for future use.

---

## 🛠 Technical Stack

Notestube is built for efficiency, speed, and accuracy:

*   **Frontend:** React, TypeScript, Tailwind CSS
*   **Backend:** Node.js (Express server)
*   **Local Persistence:** LocalStorage for Session Management & Library (History)
*   **AI Synthesis:** Advanced Generative AI API

## 💡 Use Cases

*   **Students:** Reviewing missed lectures, creating study guides for exams.
*   **Researchers:** Fast-tracking the summarization of video-based seminars and conference recordings.
*   **Lifelong Learners:** Efficiently extracting takeaways from tutorials and educational series.

## 💻 Development & Setup

### Requirements
*   Node.js (v18+)
*   NPM / Yarn

### Running the Project
1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Start the development server: `npm run dev`
